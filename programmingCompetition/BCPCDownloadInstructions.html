Taking part in the Brookfield Computer Programming Challenge? Here's what to do!

Get PC^2 the software we will be using to run the BCPC 2017.

After downloading PC^2, right click on the zip file and click on "Extract all".


Click "Next" and wait for the files to extract.


Go into the extracted folder and into the "pc2-9.4.1" folder and run the RUN_ME.bat file. If you are running OSX or Linux, you have to go into otherStuff/bin and then run "pc2client". Your username and password should have been given to you in the packet you got when you signed up.
