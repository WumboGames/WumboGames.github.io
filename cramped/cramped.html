Cramped is a puzzel platformer in which you must get back to your igloo. The only catch is that you make the level, and, following the Ludum Dare theme, you have to fit the entire level in one screen.

The controls for the game are introduced as you play-just be sure to read the directions. Want to play? You can download the jar file below and open it as long as you have Java installed.
